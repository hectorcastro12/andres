INTRODUCCION

What is the system?
- Complete management system for a bookstore
- Manages inventory, sales, and reports in an integrated manner
- Programmed in Python
- Console-based user interface with interactive menus

System Objectives:
- Manage book inventory
- Process sales with discounts
- Generate business reports
- Maintain customer records
